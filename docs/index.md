# Whetstone — AI-powered code builder

Whetstone is a developer-centric builder agent that translates simple design requests into verified, tested, and integrated codebases. It is designed to act as an autonomous pair programmer:

1. **Clarifies** ambiguous requests through interactive developer alignment.
2. **Plans** ordered dependency tasks mapping to distinct subtasks.
3. **Generates** code iterations with Worker models using past few-shot memory.
4. **Verifies** generated outputs using test-first generation, sandbox executions, and cross-model judges.
5. **Integrates** sub-task scripts into unified software artifacts.

See the [Getting Started](getting_started.md) section to begin.
